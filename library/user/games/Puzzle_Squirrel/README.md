# Puzzle Squirrel (CTF Links)

A simple Wi-Fi Pineapple Pager Games payload that displays quick links to my GitHub Pages CTFs on-screen.

## What it does
- Prints two CTF URLs to the Ctf puzzle games.

## Links shown
1. Cyber Nightmares  
   https://notorious-squirrel.github.io/Cyber-Nightmares/

2. NEXUS AI 2040  
   https://nexusai2040.github.io/NexusAi-2040/

> Note: The links to the github pages are best opened on a Laptop/Desktop (not optimised for mobile).
